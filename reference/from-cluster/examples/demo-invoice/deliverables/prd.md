# PRD · Demo Trading Co.

## 背景
# Demo 客户 · 发票自动记账

## 背景

中型贸易公司，每月约 800 张增值税发票，会计 2 人，希望 OCR + 科目推荐 + ERP 草稿，人工复核后过账。

## KPI

- 单张处理 P95 < 15s
- 字段 F1 ≥ 0.92
- 科目 Top3 命中 ≥ 0.85

## 约束

- 用友 ERP 沙箱对接
- 数据不出境，经 TKN L1


## 垂直
invoice

## 流程
OCR → 结构化 → 科目映射 → ERP 草稿（会计 HITL 复核）

## 数据边界
发票影像 L2；科目规则 L1；不含银行卡全号。
