# SOP · 法务：DPA → 合同要点 → 报价

## 触发

收到 `product-to-legal.json` 或 `ops-to-legal.json`。

## 步骤

1. **DPA**：按数据分级勾选 `dpa-checklist.md`。
2. **合同**：从 `clause-library.md` 摘条款至 `contract-summary.md`。
3. **报价**：`quote-draft.md` 含 PoC/集成/月运维三档。
4. **Handoff**：`legal-to-founder.json` → HITL `quote` / `contract`。

## HITL

Founder 确认报价与合同后方可对客户出示。

## 完成定义

- DPA 无未勾选的 PII 项
- 报价含 Token 预算区间
